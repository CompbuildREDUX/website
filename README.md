# Compbuild Static Site (GitHub Pages ready)

This is a **no-build, static HTML** version of the Compbuild page. It works out-of-the-box on **GitHub Pages**.

## Deploy steps
1. Create a new repo on GitHub.
2. Upload these files (or push via Git).
3. Go to **Settings â†’ Pages**.
   - **Source**: Deploy from a branch
   - **Branch**: `main` (or your default), **/ (root)**
4. Wait ~1 minute and visit: `https://USERNAME.github.io/REPOSITORY-NAME/`

You can edit the content in `index.html`. Assets can go into `/assets`.
